# PRD

## 제품명

- 개발명: Career Decision Calculator
- 한국어 UI명: 이직 손익계산기

## 문제

직장인은 이직, 퇴사일, 연봉협상 판단에서 연봉만 비교하기 쉽다. 실제 의사결정에는 실수령액, 상여, 복지, 교통비, 출퇴근 시간, 재택근무, 야근, 남은 연차, 퇴직금 참고액이 함께 작동한다.

## MVP 목표

사용자가 현재 직장과 후보 직장을 숫자로 비교하고, 예상 퇴사일의 금전/일정 영향을 참고할 수 있게 한다.

## MVP 기능

- 대시보드
- 이직 손익계산기
- 퇴사일 계산기
- localStorage 기반 시나리오 저장
- 캡처 가능한 결과 요약 카드

## 제외 범위

AI API, 챗봇, 서버, 로그인, 결제, 분석 SDK, 광고 SDK, 채용사이트 크롤링, 외부 캘린더 연동, 법률/노무 상담, PDF 생성, 이미지 생성, 모바일 앱, 데스크톱 앱, 브라우저 확장은 MVP에서 제외한다.

## 결과 안내 원칙

결과는 단정적 조언이 아니라 참고 계산이다. “이직하세요”, “퇴사하세요” 같은 직접 지시 문구를 쓰지 않는다.
