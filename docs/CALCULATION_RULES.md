# Calculation Rules

## 공통 안내

모든 결과는 개인 의사결정을 돕기 위한 참고 계산이다. 법률, 세무, 노무 자문이 아니다.

## 이직 비교

### 연간 현금성 보상

```txt
연봉 + 연간 상여/성과급 + 월 식대/복지 * 12 - 월 교통비 * 12
```

### 월 기준 현금성 보상

```txt
월 실수령액 또는 연봉 / 12 + 월 식대/복지 - 월 교통비
```

### 연간 출퇴근 시간

```txt
편도 출퇴근 시간 * 2 * 주당 출근일 / 60 * 연간 근무주
```

기본 연간 근무주는 52주다.

### 시간가치

기본값은 현재 연봉을 월 근무일 21.75일, 하루 8시간 기준으로 나눈 시급이다. 사용자가 직접 시급을 입력하는 구조로 확장할 수 있다.

MVP UI에서는 다음 가정값을 사용자가 수정할 수 있다.

- 연간 근무주
- 월 평균 근무일
- 시간가치 방식
- 직접 입력 시간가치
- 출퇴근 시간 가중치
- 재택근무 1일 가치
- 리스크 버퍼

### 실질 연간 이득/손실

```txt
연간 현금성 보상 차이
+ 시간가치 반영액
+ 재택근무 가치 차이
- 리스크 버퍼
```

리스크 버퍼 기본값은 후보 연봉의 10%다.

### 최소 협상 목표 연봉

```txt
후보 연봉 + 실질 손실 보전액
실질 손실 보전액 = max(-실질 연간 이득/손실, 0)
```

실질 연간 이득/손실에는 리스크 버퍼가 이미 차감되어 있다. 따라서 손실 보전액만큼 연봉을
올리면 버퍼를 반영한 채로 손익분기점에 도달한다. 여기에 리스크 버퍼를 다시 더하면 버퍼가
이중 반영되므로 더하지 않는다.

월 실수령액이 비어 있거나 0이면 월 기준 현금성 보상에서 연봉 / 12를 사용한다.
1일 통상임금이 비어 있거나 0이면 월급 정산과 연차수당에서 월 급여 / 30을 사용한다.

## 퇴사일 계산

### 근속기간

입사일부터 예상 퇴사일까지의 일수를 포함 계산한다.

### 연차수당 참고액

```txt
남은 연차 * 1일 통상임금
```

1일 통상임금이 없으면 월 급여 / 30을 사용한다.

### 월급 정산 참고액

MVP는 다음 두 가지 방식 중 하나를 선택한다.

- 30일 기준 일급: `1일 통상임금 또는 월 급여 / 30 * 퇴사월 정산일`
- 퇴사월 실제 달력일 기준: `월 급여 / 퇴사월 총 일수 * 퇴사월 정산일`

실제 급여 정산은 회사 급여기간, 휴일, 공제, 수당, 무급휴가, 지각/조퇴 기준에 따라 달라진다.

### 퇴사 체크리스트

체크리스트는 입력값에 따라 달라진다.

- 남은 연차가 있으면 연차 사용/수당 기준 확인 항목을 표시한다.
- 퇴직금 참고 계산을 포함하면 근속 1년 이상 여부에 따라 다른 확인 항목을 표시한다.
- 이직 예정일이 있으면 휴식 기간 또는 일정 겹침 확인 항목을 표시한다.
- 마지막 출근일과 퇴사일이 다르면 그 사이의 연차, 급여, 보험 처리 확인 항목을 표시한다.

### 퇴직금 참고액

근속기간이 365일 이상이면 다음 단순식을 사용한다.

```txt
월 평균 급여 * 근속일수 / 365
```

평균임금, 상여, 수당, 세금, 회사 기준을 반영한 법정 계산이 아니다.
