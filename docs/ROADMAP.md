- 브라우저 확장
- 채용공고 저장
- 채용사이트 공고 클리퍼
- Tauri 데스크톱 앱
- 모바일 앱
- Google Calendar 연동
- 퇴직금/연차수당 고도화
- 실업급여 계산
- 연봉 실수령액 세법 업데이트 자동 반영
- PDF export
- 이미지 export
- 복수 후보 회사 비교
